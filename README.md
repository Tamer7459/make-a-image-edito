# 🖼️ Image Editor Web App

### 🚀 Simple, Fast & Interactive Image Editing Tool

**Description:** A lightweight web-based image editor built with HTML, CSS, and JavaScript that allows users to apply basic filters and adjustments to images directly in the browser.

---

## 📌 Project Overview

This project is a **browser-based image editor** that enables users to upload images and apply visual adjustments such as filters and transformations without installing any software.

It demonstrates core front-end concepts like **canvas manipulation, DOM interaction, and real-time updates**.

---

## 🚀 Features

* 🖼️ Upload and display images
* 🎨 Apply visual filters (brightness, contrast, grayscale, etc.)
* 🔄 Reset image to original state
* ⚡ Real-time preview of changes
* 📱 Responsive design for mobile and desktop
* 💡 Simple and intuitive UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Canvas API

## 🧩 How It Works

1. User uploads an image from local device.
2. Image is rendered on an HTML canvas.
3. JavaScript applies filters and adjustments in real time.
4. User can modify values and instantly see results.
5. Reset option restores the original image.
